Last Visited Links

This control is no longer developed, it's replaced byt the control My links, when used as a separate control. If you already have the Last Visited Links controll added to a page it will continue to work, but you can not add any new such controls.

The Settings

To access the settings for the control, activate the Edit option and click the settings gear for the control.

This will open the settings dialog where you can set the following:

  • Row Limit: The maximum number of rows to display before a "Show more" link appears.
  • Background Color: The background color to use. If no value is set, it will fall back to the default background color set in Omnia Admin.
  • Link Color: The color to use for the links. If no value is set, it will fall back to the default text and icon color set in Omnia Admin.
  • Border Color: The color to use as border. If no value is set, no border will be applied.
  • Visible: Defines when the control should be visible: Always, up to a maximum width or to a minimum width. Then define maximum or minimum width in pixels.
